Iggyz Solana Validator

Small independently run validator

Iggyz is an independent validator operating on the Solana network, run by a Solana developer. The name is inspired by the small and powerful Italian Greyhound, whose name is often shorted to the "iggy".

What does Iggyz work on?

Staking

Native staking, including the core program and associated tooling.

Stake Pools

The SPL Stake Pool program and accompanying libraries.

Zig Solana

Zig SDK and compiler for building Solana on-chain programs.
